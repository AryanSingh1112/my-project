# Adding the Name Field:

<label for="name">Full Name:</label><br>
<input type="text" id="name" name="name" required minlength="3"
       pattern="[A-Za-z\s]+" title="Only letters and spaces are allowed"
       placeholder="Enter your full name"><br><br>

------------------------------> CODE


7. required: field must be filled in before submission.

8. minlength="3": must be at least 3 characters long.   -------------------------> EXPLANATION

9. pattern="[A-Za-z\s]": only allows letters.

10. placeholder: displays light text inside the input to guide the user.

11. type="text": allows the user to type letters.
